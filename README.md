# Описание проекта Sdaem.by

# header. Все ссылки рабочие. Ссылка квартиры на сутки позволяет выбрать город из списка + плюс прописывает в localStorage и в дальнейшем отображает выбранный город.
####

# footer. Все ссылки рабочие. Ссылки квартиры в городах переводят на Каталог квартир + изменяют город в header
####

# Главная:
#   - вкладки (Квартиры на сутки, Коттеджи и усадьбы, Бани и Сауны, Авто напрокат) рабочие, переключаются.
#   - на вкладке Квартиры на сутки, список город изначально учитывает выбор города в header, позволяет изменить выбор, не влияет на header. При манипуляции с городами каждый раз скидываются комнаты.
#   - Больше опций ведет на страницу Каталог квартир с отображенными дополнительными настройками.
#   - На карте - ссылка якорь - переводящая к блоку Поиск квартир на карте - на главной странице.
#   - Показать ведет на страницу Каталог квартир, подбирая квартиры учитывая введенные параметры + создавая  URL - ссылку

#   - Следующий блок, все карусельки рабочие, кнопочки и ссылки с городами работают по принципу кнопки Показать. Все остальные ссылки приводят на страницу с ошибкой.   

#   - Блок с карточками. При отсутствии выбранного города отображаются все квартиры Беларуси. При установке города в header появляется дополнительная опция сортировки по районам города. В случае города Минск - 2 опции сортировки по районам и по станции метро, они между собой зависимы и скидывают настройки друг друга. Карусель рабочая. Кнопка Посмотреть все ведет себя аналогично кнопки Показать.

#   - Блок карта. Кнопка Открыть карту подгружает Яндекс-карты с указанием объектов на местности и балунами. На карте имеется кнопка закрытия Яндекс-карты. Метки на Яндекс-карте зависимы от установленного в header города, а также сортировки по районам или метро. Кнопки Разместить объявление, Узнать стоимость услуги, Еще о тарифе Gold ведут на страницу с ошибкой.

#   - Блок новости. Ссылки ведут на Подробная новость, либо на отображение группы Новости в виде карточек.
####

# Новости. Домик приводит на Главную. Поиск, кнопочка Читать, Пагинация работают. Пагинация зависима от поиска.
####

# Подробная новость. Нажатие на домик приводит на Главную, на Новости - приводит на Новости.
#   - Карточки внизу страницы отображаются случайным образом и по клику Читать - обновляют страницу Подробная новость с внесением изменений + обновляет случайным образом карточки внизу.
####

# Контакты. Форма рабочая, с минимальным перечнем проверок, отображает ошибки.
# При успешных действиях выведется табличка Ваше письмо отправлено!
####

# Вход и регистрация. Выводит форму регистрации. По клику за пределами формы закрывается - переводит на ту страницу когда была нажата ссылка Вход и регистрация. Форма рабочая, с минимальным перечнем проверок, отображает ошибки.
#   - На сайте условно зарегистрирован пользователь логин и пароль admin. После ввода в левом верхнем углу header пользователь будет отображен. По клику по нему срабатывает условный выход.
#   - При клике на ссылку Создайте аккаунт появится форма Регистрация.
####

# Регистрация. По клику за пределами формы закрывается - переводит на ту страницу когда была нажата ссылка Вход и регистрация. Форма рабочая, с минимальным перечнем проверок, отображает ошибки.
#   - При отправке формы с логином admin вызовет ошибку имитируя занятость данного логина.
#   - Установлена проверка паролей обязательного повторения.
#   - При успешных действиях выведется табличка Подтвердите регистрацию.
####

# Страница ошибка 404. Кнопка Вернуться на главную приводит на Главную.
####

# Каталог квартир:
#   - при выборе комнаты - отображается плашка.
#   - Больше опций отображает скрытое меню.
#   - при отсутствии установленного города в header - в Больше опций можно выбрать только спальные места - будут отображены виде плашек и проставить галочки - в виде плашек не отображаются
#   - при установленном header городе дополнительно можно выбрать район - отобразится в виде плашки, в случае Минска - появится метро - также можно установить в виде плашки.
#   - нажатие на плашку, ее скрывает, а также изменяет критерий поиска.
#   - нажатие кнопки Показать объекты отсортирует в соответствии выбранных критерий + скроет Больше опций.
#   - нажатие кнопки Очистить - все очистит + скрывает Больше опций.
#   - Кнопка По умолчанию - рабочая, отображает два вида сортировок, сами функции сортировок отсутствуют.
#   - Форма отображения Список Плитки работают
#   - Пагинация зависимая, устанавливается при первой загрузки страницы, изменяется от применения условий кнопкой Показать объекты, нажатие на кнопки Список, Плитка.
#   - Кнопочка показать на карте - ссылка якорь - переводящая к блоку Показать найденные квартиры на карте. Блок ведет себя аналогично Блок карта на странице Главная. Изменения на карте в зависимости от условий и Пагинации
#   - Манипуляции в данном разделе формирую URL через который передаются критерии поиска. Ссылку можно использовать для сортировки.
####
